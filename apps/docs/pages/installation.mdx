# Installation

To get started with Marupay SDK, you need to install it as a dependency in your project. Follow the steps below to integrate Marupay into your application.

## Prerequisites

Make sure you have Node.js and npm (Node Package Manager) installed on your machine. You can download and install them from [https://nodejs.org/](https://nodejs.org/).

## Installing Marupay SDK

Open your project's terminal and run the following command to install Marupay SDK:

```bash
npm install marupay
