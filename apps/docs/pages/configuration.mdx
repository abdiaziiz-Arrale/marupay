# Configuring MaruPay SDK

Create a `.env` file in your project's root directory with the following content:

```.env
DAHAB_API_KEY=your_dahab_api_key
DAHAB_SECRET_KEY=your_dahab_secret_key
DAHAB_AGENT_CODE=your_dahab_merchant_id


WAAFI_API_KEY=your_waafi_api_key
WAAFI_API_USER_ID=your_waafi_user_id
WAAFI_MERCHANT_KEY=your_waafi_merchant_id
```
